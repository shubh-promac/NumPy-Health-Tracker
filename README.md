# 🏥 NumPy Health Tracker
A data science project simulating patient heart rate analysis.

### 🛠️ What I Built:
* **Synthetic Data Generation:** Created 1,000 patient records using a Normal Distribution ($\mu=70, \sigma=10$).
* **Statistical Analysis:** Calculated Mean, Median, and Standard Deviation to understand population health.
* **Outlier Detection:** Used 2 standard deviations to identify "at-risk" patients with abnormal heart rates.
* **Preprocessing:** Applied Min-Max Scaling (Normalization) to prepare data for future AI models.
